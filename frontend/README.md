# NightOwl Frontend

Initial React + TypeScript frontend scaffold for the NightOwl dashboard.

Implemented slices:

- three-panel dashboard shell
- session tree with nested children
- intent list fallback for the graph surface
- approval queue
- raw token viewer
- typed mock gateway contracts with Zod validation
- TanStack Query data bootstrapping plus mock realtime updates

Run after installing dependencies:

```bash
npm install
npm run dev
```
